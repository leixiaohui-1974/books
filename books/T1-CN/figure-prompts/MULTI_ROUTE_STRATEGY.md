## 四、统一技术规格

### HTML/SVG 图输出规范
```
- 输出为 .html 文件，浏览器截图为 PNG
- 或直接输出 .svg 文件
- 字体: Noto Sans CJK SC
- 配色: SKILL.md统一色系
- 宽度: 2400px (大图), 1800px (小图)
- 可用 puppeteer/playwright 批量截图
```

### Claude 环境可直接生成
```
- HTML artifact (.html) → 浏览器直接渲染+截图
- React JSX (.jsx) → 渲染为组件
- matplotlib → PNG直出
- Mermaid → mermaid.live 或 mmdc CLI
```
