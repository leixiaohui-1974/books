## 使用方法

1. 按章节顺序打开对应的 `chXX_figures.md` 文件
2. 将每幅图的 **Gemini 提示词** 部分复制到 Gemini
3. 检查生成图片是否符合描述要求，不满意则微调提示词
4. 将生成的图片保存为 `fig_XX-Y.png`（如 `fig_01-1.png`）
5. 插入到对应章节的 Word 文档中

## 风格统一要求

- **配色方案**：蓝色系为主调（#1E3A5F 深蓝 / #3A7BD5 蓝色 / #7CB9E8 浅蓝），安全相关用红黄绿三色
- **字体**：中文使用思源黑体/微软雅黑，英文使用 Arial
- **分辨率**：300 dpi，宽度 1024px
- **背景**：白色或极浅灰
- **线条**：2px实线，虚线用于次要关系
- **标注**：所有文字使用中文，数学符号用斜体
