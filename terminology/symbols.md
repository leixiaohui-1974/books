# CHS全体系统一数学符号表

> **维护规则**：本文件是全体系16本书的数学符号权威源。同一物理量在不同书中必须使用相同符号。

---

## 水力学基本量

| 符号 | 含义 | 单位 | 说明 |
|------|------|------|------|
| $Q$ | 流量 | m³/s | 标量，大写 |
| $h$（或 $y$）| 水位/水深 | m | $h$ 常用于水位，$y$ 常用于水深 |
| $A$ | 过水断面面积 | m² | |
| $B$ | 水面宽度 | m | |
| $S_0$ | 底坡 | — | 无量纲 |
| $S_f$ | 摩阻坡降 | — | 无量纲，Manning公式计算 |
| $n$ | Manning粗糙系数 | s/m^{1/3} | |
| $c$ | 波速 | m/s | $c = \sqrt{gA/B}$ |
| $q_l$ | 旁侧入流 | m²/s | 单位渠长入流 |

## 控制系统量

| 符号 | 含义 | 单位 | 说明 |
|------|------|------|------|
| $\mathbf{x}$ | 状态向量 | — | 粗体小写 |
| $\mathbf{u}$ | 控制输入向量 | — | 粗体小写 |
| $\mathbf{y}$ | 输出向量 | — | 粗体小写 |
| $\mathbf{A, B, C, D}$ | 状态空间矩阵 | — | 粗体大写 |
| $G(s)$ | 传递函数 | — | 拉普拉斯域 |
| $J$ | 目标函数/代价函数 | — | |

## MPC相关量

| 符号 | 含义 | 单位 | 说明 |
|------|------|------|------|
| $\tau$ | 传输延时 | s | |
| $T_p$（或 $N_p$）| 预测时域 | s（或步数）| |
| $T_c$（或 $N_c$）| 控制时域 | s（或步数）| |
| $\mathbf{Q}$ | 状态权重矩阵 | — | 注意与流量 $Q$ 区分（矩阵用粗体） |
| $\mathbf{R}$ | 控制权重矩阵 | — | |

## 安全包络量

| 符号 | 含义 | 单位 | 说明 |
|------|------|------|------|
| $\mathcal{S}$ | 安全集 | — | 花体大写 |
| $h_{min}, h_{max}$ | 水位上下限 | m | |
| $Q_{min}, Q_{max}$ | 流量上下限 | m³/s | |
| $\Delta u_{max}$ | 控制增量约束 | — | |

## 排版约定

| 类型 | 排版规则 | 示例 |
|------|---------|------|
| 标量 | 斜体 | $Q$, $h$, $n$ |
| 向量 | 粗体小写斜体 | $\mathbf{x}$, $\mathbf{u}$ |
| 矩阵 | 粗体大写斜体 | $\mathbf{A}$, $\mathbf{B}$ |
| 集合 | 花体大写 | $\mathcal{S}$, $\mathcal{X}$ |
| 运算符 | 正体 | $\max$, $\min$, $\arg\min$ |
| 上下标 | 斜体（变量）/ 正体（标识符）| $h_{max}$（正体max）, $x_i$（斜体i） |

## 各书补充符号

各专著（M1-M8）可在本表基础上扩展专用符号，但：
1. 不得与本表已有符号冲突
2. 新符号须在各书首次使用处定义
3. 建议在各书附录中列出本书专用符号表
