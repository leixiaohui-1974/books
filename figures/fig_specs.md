## 图表描述格式规范

在Markdown中用以下格式标注需要生成的图表：

```
[图X-Y: 标题]
{描述: 详细描述图表内容、布局、标注要求}
{尺寸: 半页/全页}
{颜色方案: 蓝色系/灰度/自定义}
{对应ARCH编号: ARCH-XX（如有）}
```

## 图表编号规则

- 章内连续编号：图X-1, 图X-2... / 表X-1, 表X-2...
- X = 章号
- 英文版：Figure X-1, Table X-1
- 图表编号不得跳号
