# 评审输出格式模板

请严格按以下 JSON 格式输出评审结果（用 ```json 包裹）：

```json
{
  "overall_score": 8.5,
  "dimension_scores": {
    "维度1名称": 8,
    "维度2名称": 9,
    "维度3名称": 7
  },
  "decision": "minor",
  "issues_red": [
    "🔴 致命问题1：具体描述",
    "🔴 致命问题2：具体描述"
  ],
  "issues_yellow": [
    "🟡 重要问题1：具体描述",
    "🟡 重要问题2：具体描述"
  ],
  "issues_green": [
    "🟢 建议1：具体描述"
  ],
  "comments": "总体评价（200字以内）"
}
```

## 评分锚点
- **9-10分**: 优秀，可直接发表/出版
- **7-8分**: 良好，需小幅修改
- **5-6分**: 可接受，需较大修改
- **<5分**: 不可接受，需重写

## Decision 标准
- **accept**: 无需修改，直接通过
- **minor**: 仅需小幅调整（术语、格式、个别表述）
- **major**: 需要较大修改（结构调整、补充内容、修正错误）
- **reject**: 需要重写（方向性错误、严重缺陷）
