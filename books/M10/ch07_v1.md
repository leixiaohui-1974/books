# ch07 实验六：高级异常场景编排与联合演练

## 7.1 实验目的

完成本实验后，读者应能够：

1. 设计高级异常场景编排模板并组织演练；
2. 建立场景难度分级字段并用于训练分层；
3. 制定跨班组联合演练清单并执行协同验证；
4. 评估演练效果并形成复盘改进项；
5. 为结业综合实验提供高压场景基线。

## 7.2 实验原理

高级异常场景演练强调“多故障叠加 + 多角色协同 + 时序压力”。通过可控编排异常链路，可检验组织在高压工况下的稳态恢复能力。

可定义场景处置有效性指标：

$$
SE = \delta_t T_e + \delta_r R_e + \delta_c C_e,
\quad \sum \delta_i = 1
$$

其中 $T_e$ 为处置时效得分，$R_e$ 为恢复质量得分，$C_e$ 为协同一致性得分。

[物理直觉] 异常越复杂，系统越依赖组织协同而非单点技术能力。

[工程解释] 高SE意味着“发现快、处置稳、协同顺”，可作为高等级演练通过依据。

## 7.4 场景难度分级字段（补齐 ch06 跟踪项）

建议采用三级难度：

- L1：单故障、单角色主导；
- L2：双故障耦合、跨角色协同；
- L3：级联故障、跨班组联动与时限约束。

每个场景需记录难度字段：故障数、并发度、影响范围、时限压力。

## 7.5 跨班组联合演练清单（补齐 ch06 跟踪项）

联合演练建议执行以下清单：

1. 演练前统一场景说明与角色分工；
2. 演练中实时同步关键状态与异常标记；
3. 出现升级事件时执行跨班组接管规则；
4. 演练后联合复盘并输出差异清单；
5. 明确整改责任与再演练时间。

## 7.6 实验步骤

1. 依据表7-1编排2个L2与1个L3场景；
2. 按表7-2设置难度字段并分配班组；
3. 执行联合演练并记录SE指标；
4. 对失败节点触发快速复盘与责任分配；
5. 输出“通过/补训后通过/不通过”结论。
