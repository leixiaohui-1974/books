## 图1　"多学科模型独立+系统级统一仿真验证"的底座能力示意

**Fig.1　Schematic diagram of the foundational capability of "independent multidisciplinary models + unified system-level simulation verification"**

### 核心表达意图

展示MBD方法论下"设计→运行→回馈设计"的全链条一体化闭环机制，强调四类模型（PBM、SM、OSEM、控制策略模型）各自独立建模但在系统级统一仿真验证平台上协同工作，并通过数字孪生平台实现设计期与运行期的双向贯通。

### 布局结构（建议横向布局，宽高比约 16:9）

### 视觉要素要求

- **四类模型方框**：用4个等宽矩形并排，每个内部标注中文名+英文缩写+2-3个典型方法名，用不同浅色底色区分（浅蓝PBM、浅绿SM、浅橙OSEM、浅紫控制策略）
- **统一仿真平台**：用大虚线矩形框包围四类模型，顶部横跨标注平台名称
- **场景模型**：在四类模型下方，用灰底长条矩形表示，内部分4格（水文/故障/通信/工况）
- **标准化接口**：四类模型之间用双向箭头连接，箭头旁标注"标准化状态向量接口"
- **设计阶段与运行阶段**：平台下方分左右两个圆角矩形，左蓝右绿
- **闭环箭头**：设计→运行用实线粗箭头（标注"成果交付"），运行→设计用虚线粗箭头（标注"数据回馈/再验证"），底部用弧线大箭头连接形成闭环
- **字体**：中文宋体/黑体，英文Times New Roman，标注字号不小于8pt

## 图2　CPSS框架下关键技术之间的工程逻辑关系

**Fig.2　Engineering chain between key technologies under the CPSS framework**

### 核心表达意图

展示SIM/MBD → ODD → HDC/MAS → SIL/HIL四个关键技术环节在CPSS（信息-物理-社会系统）认知框架下的工程逻辑衔接关系，强调从"仿真建模→边界定义→决策组织→在环验证"的逐级递进逻辑。

### 视觉要素要求

- **CPSS框架**：顶部用一个浅灰底大圆角矩形，内含三个并列小矩形（Cyber蓝、Physical绿、Social橙），三者用双向箭头连接表示耦合
- **四个技术环节**：用①②③④编号的四个水平长条矩形，自上而下排列，用粗箭头衔接
  - ①蓝色系（仿真建模）
  - ②绿色系（边界定义），内部用三段色带表示正常/受限/不可接受区（绿/黄/红渐变）
  - ③紫色系（决策组织），内部用两个并排子框表示HDC→MAS的演进箭头
  - ④橙色系（在环验证），内部用三级递进箭头MIL→SIL→HIL
- **衔接箭头**：各环节间的粗向下箭头旁标注信息流内容（如"模型输出+边界需求""ODD约束+场景驱动""控制策略+验证需求"）
- **CPSS到①的箭头**：表示CPSS认知驱动仿真建模
- **整体风格**：每个环节框内左侧放编号圆圈+技术名，右侧放简要内容，保持视觉对称

### 视觉要素要求

- **整体流程**：自上而下三大区域——"ODD驱动+场景生成+用例"（第一行）→"分级在环验证"（核心区域）→"五元组证据链"（底部）
- **ODD边界库**：左上角竖向三分格矩形（物理环境/运行能力/社会规程），用绿/蓝/橙三色
- **场景生成引擎**：中部齿轮图标+四种场景类型标签
- **MIL→SIL→HIL**：三个等宽矩形框横向排列，用粗箭头递进连接
  - 每个框内部分为上中下三层：被控对象层（PBM）、状态估计层（SM+OSEM）、控制策略层（HDC/MAS）
  - MIL：三层均为虚线框（纯软件），底色浅蓝
  - SIL：控制策略层变为实线框+代码图标（编译代码），底色浅绿
  - HIL：控制策略层变为深色实线框+芯片图标（真实硬件），被控对象层标注"实时仿真器"，底色浅橙
  - 每框底部用小字标注验证重点
- **五元组证据链**：底部用5个首尾相连的箭头形状框（chevron/箭头条），标注ODD→用例→指标→结果→结论
- **分叉判定**：证据链下方分为"通过→交付"和"未通过→迭代"两条路径，"迭代"路径用虚线弧线箭头回到顶部ODD边界库，形成闭环
- **控制策略模型标注**：在MIL/SIL/HIL每个框内的控制策略层中，明确标注"HDC/MAS"，体现控制模型在验证流程中的显式位置

## 通用绘图规范

| 项目 | 要求 |
|------|------|
| 输出格式 | 矢量格式（SVG/EMF/PDF），分辨率≥300dpi |
| 配色方案 | 主色调：深蓝(#1B3A5C)、中蓝(#4A90D9)；辅助色：绿(#27AE60)、橙(#E67E22)、紫(#8E44AD)、红(#E74C3C)；底色用对应色的10%-20%透明度 |
| 中文字体 | 标题：黑体；正文标注：宋体；最小字号8pt |
| 英文字体 | Times New Roman 或 Arial |
| 线条 | 主流程箭头：2pt实线；辅助连接：1pt虚线；框线：1pt实线 |
| 箭头 | 主流程用实心三角箭头；反馈/闭环用空心或虚线箭头 |
| 编号 | 框内技术环节用带圆圈的序号①②③④ |
| 标注语言 | 框内主标题中英文双语（中文在上、英文在下）；详细内容仅中文 |
| 尺寸 | 单栏宽度约8cm，双栏宽度约16cm，高度根据内容自适应 |
