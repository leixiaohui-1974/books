## 1. 概述

本指南旨在通过对用户提供的六份PPT演示文稿进行系统性分析，提炼其核心设计风格与规范，为未来制作风格统一、专业规范的演示文稿提供明确的指导和可复用的设计素材。

分析的六份PPT主题均围绕“水利”、“水网”、“智慧水务”等相关科研项目，整体呈现出高度统一的学术汇报风格。其核心特征为：**严谨、专业、信息密集、图文并茂**。设计上强调功能性，服务于信息的清晰传达，同时通过统一的视觉元素（尤其是色彩和字体）来建立品牌感和专业性。

## 2. 整体风格分析

### 2.1. 配色方案

所有PPT在配色上表现出极强的一致性，形成了稳定、专业的色彩体系。该体系以**蓝色**为主色调，象征科技、严谨与水利行业属性；以**红色**作为核心强调色，用于突出重点、标题和关键数据；辅以中性的黑、灰、白进行内容呈现。

| 角色 | 颜色 | HEX/RGB | 主要用途 |
| :--- | :--- | :--- | :--- |
| **主色调 (Primary)** | 科技蓝 | `#002060` / `#0F62AD` / `#2E75B5` | 标题、图表、背景元素、装饰性图形 |
| **强调色 (Accent)** | 中国红 | `#C00000` / `#FF0000` | 章节标题、关键词、图表高亮、关键数据 |
| **辅助色 (Secondary)** | 亮黄色 | `#FFFF00` | 图表辅助色、重点标记 |
| **辅助色 (Secondary)** | 典雅紫 | `#7030A0` | 图表辅助色、部分标题 |
| **正文/基础色** | 深邃黑 | `#000000` | 大段正文文本 |
| **正文/基础色** | 高级灰 | `#3F3F3F` | 副标题、辅助说明文本 |
| **背景/留白** | 纯白色 | `#FFFFFF` | 页面背景、文本背景 |

### 2.2. 字体使用

字体选择同样高度统一，以“微软雅黑”作为中英文内容的主力字体，保证了现代感和屏幕显示清晰度。在特定场景下，使用“Times New Roman”和“Arial”来处理纯英文或数字内容。

| 用途 | 字体族 | 建议字重 | 建议字号 (16:9) | 备注 |
| :--- | :--- | :--- | :--- | :--- |
| **主标题 (封面)** | 微软雅黑 | Bold | 36 - 44 pt | 简洁、醒目 |
| **副标题/单位** | 微软雅黑 | Regular | 20 - 28 pt | - |
| **章节/过渡页标题** | 微软雅黑 | Bold | 32 - 40 pt | 通常使用红色或蓝色 |
| **内容页标题** | 微软雅黑 | Bold | 24 - 28 pt | - |
| **正文文本** | 微软雅黑 | Regular | 16 - 20 pt | 确保阅读舒适性 |
| **图表/数据标签** | 微软雅黑 / Arial | Regular | 12 - 16 pt | 清晰、不喧宾夺主 |
| **英文/数字** | Times New Roman / Arial | Regular | 12 - 18 pt | 用于参考文献、数据表格等 |

### 2.3. 排版布局

PPT的页面结构清晰，遵循标准的学术汇报逻辑。布局上注重平衡与对齐，图文比例得当。

- **页面尺寸**: 主流为 **16:9** (宽屏)，适用于现代显示设备。仅一份PPT (`PPT5_流域水库群.pptx`) 使用4:3，在制作新PPT时应优先采用16:9。

- **页面结构**: 
    1. **封面页**: 通常包含项目名称、单位Logo（多个）、汇报人、日期。背景常使用高清的水利工程或自然景观航拍图，或深蓝色背景叠加Logo。
    2. **目录页/过渡页**: 使用“节标题”布局，以红色或蓝色大号字体呈现章节标题，起到章节分隔和引导阅读节奏的作用。
    3. **内容页**: 最常见的布局是“标题和内容”，通常采用左文右图、左图右文、上文下图等经典版式。内容区域划分清晰，逻辑性强。
    4. **结尾页**: 通常为“谢谢”或“汇报完毕”等字样，设计简洁，与封面风格呼应。

- **图文比例**: 内容页的图文比例约为 **1:1** 或 **4:6**，保证了视觉元素的丰富性和信息传达的效率。图片（包括照片、图表、示意图）是内容的重要组成部分，而非简单点缀。

### 2.4. 视觉元素风格

- **背景设计**: 封面页倾向于使用具有视觉冲击力的高清实景图片作为背景。内容页则普遍采用纯白色背景，以保证内容的可读性。

- **图表风格**: 图表类型以折线图、柱状图、饼图和流程图为主。配色遵循整体配色方案，使用蓝色系和红色系进行数据系列的区分和强调。标注清晰，易于理解。

- **图片素材**: 大量使用与项目内容相关的实景照片，如水库、大坝、河道、测控设备等。图片质量普遍较高，构图专业。此外，复杂的系统架构和技术流程则通过Visio或PPT绘制的示意图来呈现。

## 4. 设计规范与建议

1.  **坚持统一**: 严格遵循本指南定义的配色和字体规范，确保所有PPT的视觉统一性。
2.  **封面先行**: 封面是PPT的门面，应精心设计。优先选用与主题紧密相关的高清、专业航拍图作为背景。
3.  **结构清晰**: 善用“节标题”作为过渡页，将PPT内容划分为逻辑清晰的章节，引导观众思路。
4.  **图文并茂**: 避免大段纯文字页面。积极使用图表、示意图和高质量照片来辅助说明，让内容更生动、易于理解。
5.  **精选素材**: 复用 `assets` 目录中的高质量图片和示意图，避免使用低分辨率或不相关的网络图片。
6.  **保持简洁**: 保持页面元素简洁，避免不必要的装饰。设计的目的是服务于内容，而非分散注意力。
