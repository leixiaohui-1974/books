# writing-projects

微信公众号文章写作技能，用于AI辅助创作达到发布标准的高质量原创文章。

## 目录结构

```
writing-projects/
├── SKILL.md              # 核心技能文件（AI写作指令）
├── check_article.py      # 自动质量检查脚本
└── README.md             # 本文件
```

## 使用方式

### 在 Claude 中使用

告诉 Claude："帮我写一篇公众号文章，主题是XXX"，Claude 会自动读取 SKILL.md 并按流程执行。

### 在 Claude Code 中使用

将本目录放入项目的 `.claude/skills/` 或直接在 CLAUDE.md 中引用 SKILL.md 路径。

### 自动检查脚本

```bash
python3 check_article.py article.md
```

检查项包括：段落长度、句子长度、标题层级、错别字、标点符号、中英间距、加粗密度、禁用词、开头质量、结尾格式、用词重复。

## 核心设计原则

1. **手机优先**：所有排版规则围绕手机阅读体验设计
2. **零容错**：数据必须可验证，错别字必须消除
3. **五轮审查**：事实→文字→逻辑→排版→体感，逐轮不可跳过
4. **领域优化**：针对水利/CHS/HydroOS领域提供专业概念翻译表
