## 三、水网模型技能

### 3.1 水动力模型

#### 一维模型

```
控制方程:
∂A/∂t + ∂Q/∂x = 0  (连续性)
∂Q/∂t + ∂(Q²/A)/∂x + gA∂h/∂x = gA(S₀-S_f)  (动量)

数值格式:
- Preissmann 隐式格式
- MacCormack 显式格式
- TVD 格式 (总变差减小)

软件:
- HEC-RAS (免费)
- MIKE 11 (商业)
- SOBEK (商业)
```

#### 二维模型

```
浅水方程:
∂h/∂t + ∂(hu)/∂x + ∂(hv)/∂y = 0
∂(hu)/∂t + ∂(hu²+gh²/2)/∂x + ∂(huv)/∂y = ...
∂(hv)/∂t + ∂(huv)/∂x + ∂(hv²+gh²/2)/∂y = ...

数值方法:
- 有限体积法
- 有限元法
- 无网格法

软件:
- TELEMAC-2D (开源)
- MIKE 21 (商业)
- Delft3D (开源/商业)
```

#### 耦合模型

```
1D-2D 耦合:
- 河道 (1D) + 泛区 (2D)
- 管网 (1D) + 地表 (2D)

耦合方式:
- 松散耦合 (交替求解)
- 紧密耦合 (联立求解)

接口处理:
- 质量守恒
- 动量守恒
- 能量守恒
```

### 4.2 通信协议

#### Modbus

```
# Modbus RTU (串口)

帧格式:
[地址 (1)][功能码 (1)][数据 (N)][CRC(2)]

功能码:
- 01: 读线圈状态
- 02: 读离散输入
- 03: 读保持寄存器
- 04: 读输入寄存器
- 05: 写单个线圈
- 06: 写单个寄存器
- 15: 写多个线圈
- 16: 写多个寄存器

# Modbus TCP (以太网)

MBAP 头 (7 字节):
[事务标识 (2)][协议标识 (2)][长度 (2)][单元标识 (1)]
```

#### OPC UA

```
# 特点

- 跨平台 (Windows/Linux/macOS)
- 安全性 (加密/认证/授权)
- 信息建模 (对象/变量/方法)
- 发布订阅 (PubSub)

# 信息模型

节点 (Node):
- Object (对象)
- Variable (变量)
- Method (方法)
- ObjectType (对象类型)
- DataType (数据类型)

服务 (Service):
- Read (读)
- Write (写)
- Call (调用)
- Subscribe (订阅)
```

#### IEC 61850

```
# 变电站自动化标准

层次模型:
- 物理设备 (Physical Device)
- 逻辑设备 (Logical Device)
- 逻辑节点 (Logical Node)
- 数据对象 (Data Object)
- 数据属性 (Data Attribute)

通信服务:
- MMS (制造报文规范)
- GOOSE (面向通用对象的变电站事件)
- SV (采样值)
```

### 5.2 三态工作模式

```
模式一：云端主导 (Connected)
- 通信质量 > 80%
- 云端下发控制指令
- 边缘执行 + 反馈
- 控制周期：分钟级

模式二：云边协同 (Degraded)
- 通信质量 50-80%
- 云端下发策略
- 边缘实时优化
- 控制周期：秒级

模式三：边缘自治 (Isolated)
- 通信质量 < 50% 或中断
- 边缘完全自治
- 保守控制策略
- MRC 安全保底
```

### 6.2 知识抽取

```
# 从文档抽取

命名实体识别 (NER):
输入："沙坪水电站装机容量 3600MW"
输出：[沙坪水电站/工程] [3600MW/装机容量]

关系抽取 (RE):
输入："瀑布沟位于深溪沟上游"
输出：(瀑布沟，上游，深溪沟)

事件抽取 (EE):
输入："2024 年 7 月 20 日发生洪水，最大流量 11500m³/s"
输出：[洪水事件] [时间：2024-07-20] [最大流量：11500]
```

### 6.3 知识存储

```
# 图数据库

Neo4j:
- 属性图模型
- Cypher 查询语言
- ACID 事务
- 社区版免费

MATCH (n:Station)-[:MONITORS]->(s:Sensor)
WHERE n.name = '沙坪水电站'
RETURN s

# RDF 三元组

SPARQL 查询:
SELECT ?sensor ?value
WHERE {
  ?station rdfs:label "沙坪水电站" .
  ?station monitors ?sensor .
  ?sensor hasValue ?value .
}
```

### 6.4 知识应用

```
# 智能问答

问题："沙坪水电站有多少台机组？"

检索:
MATCH (s:Station)-[:HAS_EQUIPMENT]->(t:Turbine)
WHERE s.name = '沙坪水电站'
RETURN count(t)

回答："沙坪水电站共有 6 台水轮发电机组"
```
